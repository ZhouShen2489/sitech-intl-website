## Suggested Visual System

- 首页 hero：优先用 `diverse-business-meeting-room.jpg` 或 `engineers-collaborating-office.jpg`，更像“可信赖的技术合作伙伴”。
- Helport / AI 服务：优先用 `customer-support-agents-office.jpg`，如果页面偏客服中心可用 `call-center-team-workstations.jpg`。
- 运营商 / 宽带支撑：优先用 `telecom-network-switch.jpg`，故事页可用 `broadband-network-cables.jpg`。
- 数字产业 / 供应链平台：优先用 `warehouse-logistics-team.jpg`，黔牛帮故事用 `cattle-farm-platform.jpg`。
- 定制系统 / 企业门户：优先用 `software-developers-office.jpg` 或 `business-dashboard-laptop.jpg`。
- Partnership：优先用 `smiling-partnership-handshake.jpg`，CTA 可用 `partnership-handshake-reports.jpg`。
- Teamshub：优先用 `project-team-laptop.jpg`，需要移动端竖图时可用 `team-laptop-strategy-session.jpg`。
- About：优先用 `diverse-business-meeting-room.jpg`，比纯技术图更有人味。
- Hero 尽量不要重复：Home、Products、Solutions、Stories、Partnership、About、Contact 应优先使用不同图片。
